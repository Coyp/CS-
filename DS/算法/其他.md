3.洗牌(打乱数组)  
从前开始选择数字, 选好放到后面随机的下标处  

```c++  
for (int i = 0; i < res.size(); i++) { // rand() % 50 -> 随机选择 0 - 49 的数字
  int t = i + rand() % (res.size() - i); // 选择 i 后面的随机坐标
  std::swap(res[i], res[t]);
}
```

double myPow(double x, int n) {
    if (n == 0) return 1;

    double half = myPow(x, n / 2);
    if (n % 2 == 0) return half * half;
