# 双指针

- 26.删除排序数组中的重复项

    ```go
    int j = 1;
    for (int i = 1; i < nums.size(); i++) {
        if (nums[i] != nums[i-1]) { // 符合条件就交换
            nums[j++] = nums[i];
        }
        // 不符合直接略过
    }
    return j;
    ```

- 283.移动零

    ```go
    int j = 0;
    for (int i = 0; i < nums.size(); i++) {
        if (nums[i] != 0) {
            swap(nums[i], nums[j]);
            j++;
        }
    }
    ```
