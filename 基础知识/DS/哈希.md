# 哈希表

## 特性

- 查找 O(1) -> 应用: 查找
- 负载系数 = 元素个数 / 表格大小

## 解决碰撞方法

- 线性探测
  - 遇到冲突直接往后放 H + 1, H + 2 ....
- 二次探测
  - 遇到冲突之后 H + 1平方, H + 2平方...
  - 表格大小为质数且负载系数保持在 0.5 以下, 可以保证新元素探测次数不多于 2
- 开链定址
  - 碰撞后使用 list 往后连接
