# sort

## quick sort

```c++
// 此版本减少了把最后换回中间的操作
int pivot = arr[end];
int first = 0;
int last = arr.size();

while (true) {
    while (arr[first] < pivot) first++;
    last--;
    while (pivot < arr[last]) last--;
    if (!(first < last)) break;
    std::swap(arr[first], arr[last]);
    first++;
}
交错, 跳出循环后
last 指向 L 的最后一个值
first 指向 R 的开头
```
